# Mollagoli Printing House

پروژه توسعه و مدیریت سیستم چاپخانه ملاگلی. این ریپازیتوری شامل کدهای منبع و مستندات فنی مربوط به پیاده‌سازی این پلتفرم است. هدف اصلی این پروژه، ارائه یک زیرساخت پایدار با رابط کاربری کاملاً مینیمال، مدرن و بهینه برای مدیریت فرآیندها است.

## ویژگی‌های کلیدی
- **معماری ساختاریافته:** توسعه یافته با تمرکز بر توسعه‌پذیری و نگهداری آسان (Clean Architecture).
- **رابط کاربری (UI/UX):** طراحی مدرن و به دور از پیچیدگی‌های بصری (پیاده‌سازی شده با استانداردهای روز و تایپوگرافی خوانا نظیر فونت یکان‌بخ).
- **عملکرد بهینه:** ساختار سبک برای پردازش سریع‌تر درخواست‌ها و بارگذاری سریع صفحات.
